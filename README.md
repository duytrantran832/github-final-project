# Simple Interest Calculator

A simple tool to calculate future interest payments based on the principal amount, annual interest rate, and time period.

## Description
This project is a web-based (hoặc script-based) application that allows users to quickly determine the interest earned or owed on a loan or investment using the simple interest method.

## Formula
The calculator uses the standard simple interest formula:

$$A = P(1 + rt)$$

Where:
* **A**: Total amount (Principal + Interest)
* **P**: Principal amount
* **r**: Annual interest rate (decimal)
* **t**: Time period (in years)

To calculate interest alone:
$$I = P \times r \times t$$

## How to Use
1.  Enter the **Principal amount**.
2.  Input the **Annual Interest Rate** (e.g., 5 for 5%).
3.  Specify the **Time period** in years.
4.  Click "Calculate" to see the results.

## Example
* **Principal:** $1,000
* **Rate:** 5%
* **Time:** 2 years
* **Result:** $100 in interest ($1,100 total)
